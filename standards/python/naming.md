# Python — Naming

- `snake_case` for functions, variables, modules. `PascalCase` for classes. `UPPER_CASE` for constants.
- Full words, no abbreviations: `calculate_expected_value()` not `calc_ev()`, `session` not `db` (when it's a session), `manager` not `mgr`.
- Boolean names read as predicates: `is_active`, `has_access`.
- Module names are short, lowercase, no underscores where avoidable (`payments`, not `payment_utils_helpers`).
